## What is figma-mcp?

`figma-mcp` is a Model Context Protocol (MCP) server that pairs your AI client to Figma. It runs as an `npx`-installable binary that any MCP-aware client (Claude Code, Claude Desktop, Cursor, Windsurf, VS Code Copilot) can launch on demand. The pairing happens through a small bridge plugin that you drag-import into Figma Desktop once — after that, the AI can read your selection, query variables, and mutate the canvas.

The server ships with roughly 25 tools today, grouped into packs for extracting node data, reading and writing variables, capturing the bridge plugin's console output, and mutating the canvas. The catalog grows over time; your AI client discovers tools via the standard `tools/list` handshake, so there's nothing to memorize.

## Install

```bash
npx @bromso/figma-mcp setup
```

`setup` auto-detects the AI clients installed on your machine, prints a table of the config files it will write, and writes them atomically. Re-running is idempotent — sibling MCP entries are preserved. See the [Quickstart](/docs/getting-started) for the full first-run flow.

## Verify

```bash
figma-mcp doctor
```

`doctor` reports daemon liveness, plugin pairing, AI-client config drift, recent daemon errors, and IPC socket conflicts. A clean run prints all five checks as `ok`.
